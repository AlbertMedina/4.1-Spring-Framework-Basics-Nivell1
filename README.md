# Introduction to Spring Boot

## Description - Exercise statement

### Level 1
Creating a basic REST API using Spring Boot that responds to HTTP requests. Implement a health check endpoint to verify the application is running and returns JSON status.

### Level 2
Managing an in-memory list of users with endpoints to create, retrieve, and filter users.

### Level 3
Refactoring the application into a layered architecture with Controller, Service, and Repository. Applying unit testing with Mockito for service logic and integration testing for the full API.

## Technologies used
- Oracle OpenJDK 21.0.8
- Spring Boot 3.5.8
- Maven 3.9.11
- Postman
- IntelliJ IDEA Comunity Edition

## Requirements
- Oracle OpenJDK 21.0.8
- Spring Boot 3.5.8
- Maven 3.9.11
- Java IDE

## Installation
1. Clone repository (https://github.com/AlbertMedina/4.1-Spring-Framework-Basics-Nivell3.git).
`git clone https://github.com/AlbertMedina/4.1-Spring-Framework-Basics-Nivell3.git`
2. Import project to Java IDE.

## Execution
1. Open project with Java IDE.
2. Locate UserApiApplication.java and run it from your IDE (or run `mvn spring-boot:run` in your terminal).
3. Use your browser or postman to test the endpoints.
4. Tests can also be found in the test folder.
